# PerformanceGraph
---
*Create a performance graph from a file created from log of the time command*

To create the graphical view of the data contained in the file 

`./graphic <file> <output.pdf>`

* The file structure should be as:

```

    real  0m0.030s
    user  0m0.024s
    sys 0m0.004s
    
    real  0m4.022s
    user  0m2.896s
    sys 0m0.124s
    
    real  0m4.037s
    user  0m2.888s
    sys 0m0.140s
```
